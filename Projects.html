Scratch Project


INSTRUCTIONS:

Up, Left, Right Arrow Keys - Move Blue Wizard

W, A, D - Move Red Wizard

Press the Stop Sign then the Green Flag to start the game

OBJECT OF THE GAME: This is a two player game where two people compete to collect as many coins as possible.

There are three rounds and after each round the time the coin stays in place decreases and the map changes.

There is a one minute timer for every round and the player with the highest score wins the game.

With a partner, I was able to create a 2D game that involved many features such as time, levels, and score. Both of us contributed equally to the project and were able to work together to develop an idea we had and debug our code as much as possible.
